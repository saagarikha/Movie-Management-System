# Movie-Management-System
This project is basically done to illustrate my understanding and skill-set of Object Oriented Analysis and Design domain knowledge. This is an end to end project with a Model - View - Controller Framework. Model being written in JSP and servlets saving the purpose of executing the Controller and View being HTML. After a ticket is booked, an email is being sent to the user regarding the confirmation. A mock payment gateway is established to demonstrate this as a standard working protocol. E-mail servlets in Java is used. Database is a MySQL database for storing the uesr details. AJAX concept is used in the view to ease the UI and reduce the overlay for good user experience.
